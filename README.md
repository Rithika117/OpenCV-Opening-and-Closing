## Aim

To perform Opening and Closing operations using OpenCV.

## Software Required
Python
OpenCV
NumPy
Jupyter Notebook / VS Code
## Steps
Step 1:

Import the required packages.

Step 2:

Create or read the input image.

Step 3:

Create the structuring element.

Step 4:

Apply Opening operation.

Step 5:

Apply Closing operation.

Step 6:

Display the output images.

## Output
The input image is displayed first. Opening operation removes small noises from the image, while Closing operation fills small holes and gaps. The processed output images are displayed after applying both operations.
## Result

Thus, Opening and Closing operations were successfully performed using Python and OpenCV.
